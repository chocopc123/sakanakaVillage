# 🔰 ウェブサイト更新マニュアル

このマニュアルでは、ウェブサイトの更新手順を分かりやすくご案内します。
プログラミングの知識は一切不要ですので、ご安心ください。

---

## 🌟 更新の流れ

このウェブサイトは、**「データの入力」** と **「組み立て（ビルド）」** の 2 ステップで出来上がります。
難しいコードを直接触る必要はありません。メモ帳感覚で文字を書き換えるだけで OK です！

1.  **データを編集する**：ニュースやイベント情報を書き換えます。
2.  **組み立てボタンを押す**：書き換えたデータを元に、自動でホームページを作ります。
3.  **公開する**：出来上がったファイルをサーバーにアップロードします。

---

## 📁 フォルダの役割

まずは、フォルダの中身を簡単に説明します。よく使うのは **`data`** と **`src`** の 2 つだけです！

```text
sakanakaVillage/
├── data/            👈 【ここを編集します】
│   ├── topics.json      (トップページのトピックス)
│   ├── news.json        (トップページのニュース)
│   ├── events.json      (大会情報ページ)
│   └── ...
│
├── build.bat        👈 【組み立てボタン】
│                        (これをダブルクリックするとホームページが更新されます)
│
└── src/             👈 【完成品】
    ├── index.html       (ここにあるファイルがサーバーにアップロードされます)
    └── ...
```

---

## ✏️ 手順 1：データを編集しよう

`data` フォルダの中にあるファイル（.json ファイル）を編集して、内容を書き換えます。

### ⚠️ 編集するときのお約束（重要）

これらのファイルは「プログラムが読むためのメモ」なので、以下の記号を誤って消さないように注意してください。

- `"` (ダブルクォーテーション)
- `,` (カンマ)
- `[` `]` `{` `}` (カッコ)

**もし消してしまっても大丈夫！** 元に戻すか、分からなくなったら開発者にご連絡ください。

---

### 各ファイルの書き方ガイド

#### 1. ニュースを更新する (`news.json`)

トップページの「NEWS」欄を更新します。

```json
[
  {
    "date": "2025.12.11", // 日付
    "title": "更新タイトル", // 記事のタイトル
    "description": "記事の内容" // 短い説明文
  }
]
```

#### 2. 大会情報を更新する (`events.json`)

「大会情報」ページを更新します。

```json
[
  {
    "date": "2023/11/12", // 大会の日付
    "name": "大会の名前", // 大会名
    "pdf": "events/taikai1.pdf", // PDFファイルの名前（src/events/フォルダに入れてください）
    "videoId": "pD7AAttA6CY" // YouTubeの動画ID (URLの v= の後ろの部分です)
  }
]
```

#### 3. トピックスを更新する (`topics.json`)

トップページのカードを更新します。

```json
[
  {
    "icon": "two_wheeler", // アイコンの名前 (Google Fonts Iconsから選べます)
    "title": "レンタルバイク", // タイトル
    "description": "説明文です。<br>改行したいときは<br>と書きます。",
    "badge": "注目!", // 右上の「注目！」などのリボン（不要ならそのままでOK）
    "campaign": "false" // "true" にするとキラキラ光ります
  }
]
```

#### 4. 会社概要を更新する (`company.json`)

「施設概要」ページの情報を更新します。

```json
[
  {
    "label": "施設名",
    "value": "坂中村トライアルランド (SAKANAKA VILLAGE)"
  },
  {
    "label": "管理者",
    "value": [
      // 複数の人の名前を入れたいときは、このようにリストにします
      {
        "name": "中村 篤",
        "machine": "TRS XTRAC RR-E 250"
      }
    ]
  }
]
```

#### 5. 沿革を更新する (`history.json`)

「沿革」ページの情報を更新します。

```json
[
  {
    "date": "2023年9月",
    "description": "内容の説明です。<br>ここはHTMLタグが使えます。"
  }
]
```

---

## 🔨 手順 2：ホームページを組み立てよう（ビルド）

編集が終わったら、変更を反映させましょう。

1.  フォルダの中にある **`build.bat`** というファイルを**ダブルクリック**してください。
2.  黒い画面が一瞬出てきて、すぐに消えます。
3.  これで更新完了です！

※ エラーが出る場合は、手順 1 で記号 (`"` や `,`) を消してしまっていないか確認してみてください。

---

## 🚀 手順 3：インターネットに公開しよう

最後に、更新されたファイルをサーバー（インターネット上の置き場所）にアップロードします。

**アップロードするのは `src` フォルダの中身だけです！**

### ✅ アップロードするもの（`src`の中身全部）

- `index.html`
- `events.html`
- `about.html`
- `style.css`
- `script.js`
- `images` フォルダ
- `events` フォルダ

### ❌ アップロードしなくていいもの

- `data` フォルダ
- `build.bat`
- `build.py`
- `README_BUILD.md` (このマニュアル)

これで完了です。お疲れ様でした！✨

---

## ❓ 困ったときは？

- **Q. 更新したのに変わらない！**
  - A. `build.bat` をダブルクリックしましたか？
  - A. ブラウザの「再読み込み」ボタン（またはキーボードの F5 キー）を何度か押してみてください。
- **Q. 画面が真っ白になった / エラーが出た**
  - A. 編集したファイルの `"` や `,` が抜けているかもしれません。確認してみてください。

もしどうしても解決しない場合は、いつでも開発者までご連絡ください！👨‍💻
